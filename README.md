# Serviworld Logistics Website

Sitio web de Serviworld Logistics construido con Astro y Kinto CMS.

## Tecnologías

- **Astro** - Framework estático
- **Kinto CMS** - Sistema de gestión de contenido
- **Tailwind CSS** - Estilos
- **Cloudflare Tunnel** - Deploy

## Estructura

```
serviworldlogistics/
├── sites/serviworldlogistics/  # Sitio Astro
├── skills/                     # Skills de Kinto CMS
└── config/                    # Configuración
```
